# Smirverb
Packaging Signalsmith Audio's [excellent FDN reverb](https://github.com/Signalsmith-Audio/basics/blob/main/include/signalsmith-basics/reverb.h) into a JUCE Plugin so my friend Dan Smirnov can use it in his DAW.

### Using:
After cloning, use `init.sh` to configure your build environment, and `run.sh` to build.
